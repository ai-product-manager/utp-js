Eventos en JavaScript – Demo Interactiva

Escucha y reacciona a acciones del usuario o del navegador. Usando addEventListener()

Ejemplos

1) onclick

Detecta un clic y cambia el color del botón.

2) onchange

Registra cuando cambias la opción seleccionada.

3) onfocus / onblur

Marca el input al enfocarlo y limpia al salir.

4) onkeydown / onkeyup

Muestra la tecla presionada/soltada.

Tip: prueba con letras y números

5) onmouseover / onmouseout

Pasa el mouse por el recuadro.

Hover aquí

6) onselect

Selecciona parte del texto para disparar el evento.

7) onsubmit

Previene el envío y valida contenido.

Se cancela el envío real con event.preventDefault()

8) onload

Se registra cuando el documento está listo.

Revisa el panel de logs para ver el evento al cargar.
