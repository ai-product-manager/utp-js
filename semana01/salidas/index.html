JavaScript Básico – Demo Interactiva

Ejemplos de document, alert / confirm, prompt y console.

Ejemplos

1) document.getElementById()

Accede y modifica un elemento del DOM.

Soy #target

2) document.write() / writeln()

Demostración en una ventana nueva para no reemplazar esta página.

Nota: document.write() tras la carga de la página reemplaza el documento.

3) alert()

Muestra un cuadro de diálogo informativo.

4) confirm()

Pregunta al usuario (OK/Cancelar) y retorna true/false.

5) prompt()

Pide un valor al usuario; retorna texto o null.

6) console.*

Registra mensajes en la consola del navegador.

Abre las herramientas del navegador (F12) y ve la pestaña Console.
